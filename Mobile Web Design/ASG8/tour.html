Assignment 5

In this assignment we take a turn now to create 2 different tables and a grid:
  • Table1 includes a reflow table displaying at least 5 state names with thier associated state capital.

  • Table2 includes a column toggle table showing the same 5 states, along with the state capital, state bird, state flower, and state animal.

  • Grid1 includes a responsive grid showing the state name, capital, bird, and flower in a four grid layout. Also beneath each, a single sentence (paragraph) so that the text wrapping will show the column size of each column.
